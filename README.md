# Anantdv GitHub Pages Static Site

This folder is the GitHub Pages-ready static version of the Frappe website.

## Files
- index.html
- why-anantdv.html
- about-us.html
- contact-us-adv.html
- assets/anantdv_branding/images/

## Important
Copy your real image/video files into:
assets/anantdv_branding/images/

Required files include examples like:
- anantdv-logo.png
- anantdv-hero.png
- Video1_09x.mp4
- erp-3d.png
- ai-3d.png
- inf-3d.png
- web-3d.png
- erp1-3d.png through erp5-3d.png

## Deploy
Upload all files/folders to a GitHub repository, then enable GitHub Pages from Settings > Pages > Deploy from branch > main > /root.

## Note
The contact form is static. Connect it to Formspree, Google Forms, or another backend if you need submissions.
